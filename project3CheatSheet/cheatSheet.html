HTML Cheat Sheet

Tag Name Description
<h1> </h1> Headings Inserts a heading text and goes from h1 to h6
<p> </p> Paragraph Inserts a default paragraph text
<br> </br> Line Break Inserts a line break
<ol> </ol> Ordered list Creates an ordered list, but without the items
<ul> </ul> Unordered list Creates an unordered list, but without the items
<li> </li> List item Creates a list item within an ordered list or an unordered list
<img src='' alt=''> Image Inserts an image
<video src='' controls> video not supported </video> Video Inserts a video provided by the source. You can put a text inside in case the video doesn't work properly
<embed src='' type='application/pdf'> Embed Inserts external contents like PDFs, videos or interactive media
<div> </div> Division It's just a division
<span> </span> Span Creates a division within the text. Helpful when trying to modify a group of words within a text
<em> </em> Emphasis Element / Italics Turns the text inside into italics
<strong> </strong> Strong Element / Bold Turns the text inside into bold
HTML Structure
<head> </head> Head Everything related to the intern structure of the html file
<body> </body> Body Holds the visual content within the html page
